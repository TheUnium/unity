About me

Are you tired of adding lots of bots in each server each doing separate things? Well, introducing : Unity! Unity is a discord bot which has every feature you might possibly need. With more than 300 commands, including the ability to make custom commands, the sky is your limit! And of course, everything is absolutely free (but donations are appreciated üòÑ)!


Moderation

Unity has 17 advanced moderation commands. This contains basic moderation commands such as : /moderation ban, /moderation kick, /moderation unban, /moderation clear, /moderation timeout and some more advanced commands such as : /moderation tempban, /moderation softban, /moderation nuke, /moderation lockdown, /moderation warn, /moderation demote, etc.


Custom Commands

Unity supports custom commands! You can add and delete custom command. For now it is not extremly advanced, it only supports sending embeds, rugular messages and DMs. You can add custom commands by doing : /custom-commands add command:[command_name] text:[what_the_command_returns] and then follow the setup embed and delete by running : /custom-commands delete command:[command_name]


Fun

I am not going to into too much detail on this one, but this bot includes a bunch of fun commands accessible by running /fun which includes memes, user commands (like /fun user hack), text commands which modify how text look (like turning them into emojis) and extra commands like astronomy picture of the day (/fun extra apod)!


Soundboard

Unity contains a soundboard for all your memeing desires! This contains nostalgic sounds from Windows, a few popular meme songs like Never going to give you up - Rick Astley, Roundabout - Yes (The to be continued meme), etc and of course, actual memes like FBI OPEN UP!!11!, Mission failed., etc.


Giveaways

Unity contains giveaways. You can /giveaway start, /giveaway stop, /giveaway drop, /giveaway end, /giveaway edit, /giveaway delete, /giveaway pause, /giveaway unpause and /giveaway reroll. This is everything that you might need when doing giveaways!


Economy

Unity includes advanced economy which has everything you might need! Inspired by unbelievaboat's economy system, you can add items, add money to users, beg for money, work for money and also rob users with this commands respectively /economy additem, /economy addmoney, /economy beg, /economy work, /economy rob, etc.


Ticket system

Do you own a support server or something similiar to it? Well unity has the solution! With a ticket system just as good as ticket bot, you can easily it up with just 2 commands. You can easily add users to a ticket, remove a user from a ticket (other than the person who created the ticket), claim a ticket, unclaim a ticket, etc. When you close a ticket you automaticly get a option to reopen the ticket, delete the ticket channel and of course download a transcript.


Sticky messages

Going to keep it short and simple, you can run /stickymessages stick channel:[channel] message:[message] to stick a embed which contains the text you specified in the message variable. There can be only 1 message stuck to the bottom of any channel (for obvious reasons). To untick a message you can run /stickymessages unstick channel:[channel] which will unstick any message from that channel (that is already a sticky message). You can also list all the stick messages by running /stickymessages messages.


These are not the only things in this bot of course, there are hundreds of more features but that will be too long for this description. Best part? No pay walls. Developed by Captain McD (me!) and my school friend Unium (The main developer).
